# Contacts App (Тестовое задание - Вариант 1)

## О приложении

Тестовое задание для Android-разработчика. Приложение для просмотра контактов устройства с возможностью совершения звонков.

## Технологический стек

- **Язык**: Kotlin
- **UI**: Jetpack Compose (Material 3)
- **Архитектура**: 
  - Clean Architecture
  - MVVM
- **Библиотеки**:
  - Dagger Hilt (DI)
  - Coroutines + Flow
  - Coil (загрузка изображений)

## Функционал

### Основные возможности:
✅ Просмотр списка контактов устройства  
✅ Группировка контактов по первой букве имени  
✅ Отображение:
- Имени и фамилии
- Основного номера телефона
- Аватарки контакта (если есть)

✅ Совершение звонка по нажатию на контакт  
✅ Кнопка для обновления списка  
✅ Обработка ошибок и состояний загрузки  


## Запуск проекта

1. Клонировать репозиторий
2. Открыть в Android Studio
3. Запустить на эмуляторе или устройстве (minSdk 26)

## Особенности реализации

- Используется ContentResolver для доступа к контактам
- Оптимизированные запросы к ContactsContract
- Состояния UI: Loading/Success/Error
- Полная поддержка Jetpack Compose
