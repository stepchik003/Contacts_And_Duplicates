# Contacts App (Тестовое задание - Вариант 3)

## О приложении

Тестовое задание для Android-разработчика. Приложение для просмотра контактов устройства с возможностью совершения звонков и удалением дублирующихся записей.

## Технологический стек

- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **Архитектура**: 
  - Clean Architecture (разделение на Domain, Data, Presentation)
  - MVVM
- **Библиотеки и компоненты**:
  - Dagger Hilt
  - Coroutines + Flow
  - Coil
  - Android IPC (AIDL, Bound Services)

## Функционал

### Основные возможности:
- Просмотр списка контактов устройства
- Группировка контактов по первой букве имени
- Совершение звонка по нажатию на контакт
- **Поиск и удаление дублирующихся контактов (UserStory 2)**
- Обработка Runtime-разрешений
#### Отображение:
- Имени и фамилии
- Основного номера телефона
- Аватарки контакта (или заглушки)
